основы анализа рынка

Некоторые основные индикаторы рынка:

Трендовые индикаторы. Используются для выявления трендов в движении цены.
Например, скользящая средняя показывает усреднённые цены за конкретный период.
Осцилляторы.
Исследуют рыночную активность одновременно
с прочими факторами, которые изменяются в течение определённого времени.
Например, индекс относительной силы (RSI) позволяет оценить перекупленность или перепроданность актива.
Индикаторы объёмов.
Включают в себя алгоритмы, которые анализируют объёмы торгов по торгуемым активам
Например, вертикальные объёмы показывают количество сделок за определённый период,
а горизонтальные — количество сделок с активом по определённой цене.
Информационные индикаторы.
К ним относятся экономический календарь, виджеты, которые содержат сводки новостей,
сведения об открытых сделках и выставленных ордерах.
